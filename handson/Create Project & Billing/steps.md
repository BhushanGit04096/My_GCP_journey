# Create Project & Enable Billing (Console Steps)

---

## Step 1: Open Google Cloud Console

1. Go to https://console.cloud.google.com
2. Sign in using your Google account

---

## Step 2: Create a New Project

1. Click the **Project Selector** (top-left)
2. Click **New Project**
3. Enter:
   - **Project Name**: gcp-handson-project
   - **Organization**: Leave default (if personal)
   - **Location**: Default
4. Click **Create**

## Step 5: Link Billing Account to Project
