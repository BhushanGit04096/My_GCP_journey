# Identity & Access Management (IAM)

This section covers **Identity & Access Management (IAM)** in Google Cloud Platform.
IAM controls **who can do what on which resource**.

In this hands-on, we focus on:
- Using **predefined IAM roles**
- Creating and using **service accounts**
- Binding IAM roles at the **project level**

This is a **core Cloud Engineer skill** and heavily tested in interviews.
